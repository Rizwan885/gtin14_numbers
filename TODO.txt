Step 1: if there is no vendor folder run "Composer install" to install pacakges defined in composer.json file. 


Step 2: In Browser open index.php.

localhost/foldername/index.php

Example:  http://localhost/gtin/index.php


Step 3: In Form Input enter range limit e.g "10,000" in input field and click on button. It will check file exist or not.
If file exists it will remove all contents and add new numbers. If does not exist it will create new file and add numbers.
